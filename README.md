# MyFinanceTraining

Global Event → Japan Market Reaction を対象に、ニュース・市場イベントから連想ツリーを生成し、日本株/ETF/ETNの投資仮説を管理・検証するための個人向け投資仮説OSです。

## 目的

このリポジトリでは、投資判断そのものをAIに丸投げするのではなく、次の流れを仕組み化します。

```text
ニュース/市場イベント
→ 直接影響
→ 企業・生活者の行動変化
→ 需要/供給/コスト/価格変化
→ 上がる業種・下がる業種
→ 候補銘柄/ETF/ETN
→ 過去検証
→ 採用/保留/棄却
```

## 基本方針

- 対象市場は初期版では日本株・日本ETF/ETNに絞る
- 入力情報は世界のニュース、米国市場、金利、為替、商品、地政学イベントを扱う
- AIは常時利用せず、深掘り・反論・未登録テーマの連想生成だけに限定する
- 通常処理はルールベース、銘柄候補抽出、スコアリング、仮説DBで行う
- 売買自動化は初期スコープ外とし、判断補助・検証・記録に限定する

## アプリ起動

```bash
npm install
npm run dev
```

ビルド確認：

```bash
npm run build
npm run preview
```

GitHub Pages 配信を想定し、Viteの `base` は `/MyFinanceTraining/` に設定しています。

## 初期スコープ

- ニュース/イベントの手入力
- 5段階の連想ツリー管理
- 仮説ID、根拠、期待方向、失敗条件の登録
- 日本株テーママスタの整備
- ルールベースの候補抽出
- 必要時のみAIで深掘り
- CSVベースのバックテスト準備
