# run

Start the MyFinanceTraining dev server and verify it works.

## Steps

1. Run `npm run dev` in the background
2. Wait a moment for the server to start
3. Open http://localhost:5173/MyFinanceTraining/ via browser snapshot
4. Verify the sidebar navigation and Dashboard view load correctly
5. Report any errors to the user

## Notes

- Base URL: `/MyFinanceTraining/` (set in vite.config.ts)
- Dev server default port: 5173
- localStorage key: `myfinancetraining_v1`
