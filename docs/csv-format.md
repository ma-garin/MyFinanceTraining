# CSV バックテスト フォーマット定義

## 概要

バックテストは2種類のCSVを入力として使用する。

- **価格データ CSV** — 銘柄ごとの日次終値
- **イベント CSV** — 仮説IDとイベント発生日・対象銘柄の紐づけ

## 1. 価格データ CSV (`prices.csv`)

### フォーマット

```csv
date,ticker,close
2024-06-03,7974.T,7800.0
2024-06-04,7974.T,7950.0
2024-06-05,7974.T,8100.0
```

### カラム定義

| カラム | 型 | 説明 |
|---|---|---|
| `date` | YYYY-MM-DD | 営業日の日付 |
| `ticker` | string | 銘柄コード（例: `7974.T`, `1570.T`） |
| `close` | float | 終値（調整済み推奨） |

### 注意事項

- ヘッダー行必須
- 非営業日（土日・祝日）は含まなくてよい
- 同一 ticker は日付順に並べることを推奨（必須ではない）
- 調整済み終値（株式分割・配当考慮）の使用を推奨

## 2. イベント CSV (`events.csv`)

### フォーマット

```csv
hypothesis_id,event_date,ticker,notes
MIDEAST-OIL-UP-STAYHOME-GAME-UP,2024-06-03,7974.T,任天堂
MIDEAST-OIL-UP-STAYHOME-GAME-UP,2024-06-03,9766.T,コナミG
US-JOBS-STRONG-AI-DOWN-JP-RETAIL-UP,2024-06-07,3382.T,セブン&アイ
```

### カラム定義

| カラム | 型 | 説明 |
|---|---|---|
| `hypothesis_id` | string | 登録済み仮説IDまたは任意の識別子 |
| `event_date` | YYYY-MM-DD | イベント発生日（この日の終値を基準価格とする） |
| `ticker` | string | 検証対象銘柄コード |
| `notes` | string | 備考（任意）|

### 注意事項

- 1行 = 1仮説 × 1銘柄
- 複数銘柄を検証したい場合は行を複数追加する
- `event_date` が非営業日の場合、その日の価格データが存在しないため `—` と表示される

## 3. 検証指標

### リターン計算

- **T+1**: イベント翌営業日終値 / イベント日終値 − 1
- **T+3**: イベント後3営業日終値 / イベント日終値 − 1
- **T+5**: イベント後5営業日終値 / イベント日終値 − 1

営業日カウントは土日をスキップする（祝日は考慮しない）。

### サマリー指標

| 指標 | 説明 |
|---|---|
| **勝率** | リターンが正（> 0）の割合 |
| **平均リターン** | 全データの算術平均 |

### 解釈ガイド

- 勝率 > 60% かつ平均 T+5 > 2% → 仮説として有望、深掘り検討
- 勝率 < 40% → 方向性を再検討
- サンプル数 < 5 → 統計的有意性が低い、参考値として扱う

## 4. データ取得先（参考）

日本株の価格データは以下から取得可能（各サービスの利用規約に従うこと）。

| ソース | 用途 |
|---|---|
| Yahoo Finance Japan | 個別銘柄の日次終値 |
| JPX（日本取引所） | 指数・業種別データ |
| stooq.com | 無料の日次CSVダウンロード |
| kabuoji3.com | 日本株終値CSV |

## 5. サンプルデータで確認できること

以下のサンプルを価格CSVとイベントCSVに貼り付けると、T+1〜T+5のリターン計算を体験できる。

**価格CSV（任天堂 7974.T の6営業日分）:**

```csv
date,ticker,close
2024-06-03,7974.T,7800.0
2024-06-04,7974.T,7950.0
2024-06-05,7974.T,8100.0
2024-06-06,7974.T,8050.0
2024-06-07,7974.T,8200.0
2024-06-10,7974.T,8180.0
```

**イベントCSV:**

```csv
hypothesis_id,event_date,ticker,notes
MIDEAST-OIL-UP-STAYHOME-GAME-UP,2024-06-03,7974.T,任天堂
```

**期待される結果:**

| 指標 | 計算 | 結果 |
|---|---|---|
| T+1 | 7950 / 7800 − 1 | +1.9% |
| T+3 | 8050 / 7800 − 1 | +3.2% |
| T+5 | 8180 / 7800 − 1 | +4.9% |
