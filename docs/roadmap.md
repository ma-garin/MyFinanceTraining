# ロードマップ

## v0.1: 投資仮説DB

目的：手入力で仮説を構造化して保存できる状態を作る。

- README/設計ドキュメント整備
- イベント入力
- 連想ツリー登録
- 仮説ID登録
- 採用/保留/棄却ステータス
- 日本株テーママスタ

## v0.2: PWA基盤

目的：スマホ/PCの両方で使えるWebアプリ基盤を作る。

- React + Vite 初期化
- PWA設定
- レスポンシブUI
- IndexedDB保存
- GitHub Pagesデプロイ準備

## v0.3: ルールベース連想

目的：AIなしで、主要ニュースから候補テーマを出せるようにする。

- キーワード辞書
- イベント分類
- 連想テンプレート
- テーマ候補提示
- 失敗条件テンプレート

初期テンプレート候補：

| テーマ | 連想例 |
|---|---|
| 中東情勢 | 原油、防衛、商社、海運、金、ゲーム、旅行警戒 |
| 米雇用統計 | 金利、AI/半導体、小売、銀行、REIT |
| SOX急落 | 日本半導体警戒、小売/内需ローテーション |
| 原油急騰 | 石油、商社、防衛、空運警戒、外出抑制 |
| 円安 | 輸出、インバウンド、輸入コスト、小売選別 |

## v0.4: AI深掘り

目的：必要なときだけAIで連想を増やす。

- AI深掘りボタン
- 実行上限管理
- 結果キャッシュ
- 反対シナリオ生成
- 見落とし観点生成

## v0.5: CSVバックテスト準備

目的：仮説を検証できる土台を作る。

- CSVインポート
- 指数/業種/銘柄データの取り込み
- イベント日と翌営業日リターンの照合
- 勝率、平均リターン、最大下落率

## v0.6: レポート

目的：仮説の成績と判断履歴を見える化する。

- 仮説別成績
- テーマ別成績
- 採用/棄却履歴
- 月次レビュー
- Markdownエクスポート

## v1.0: 個人向け投資仮説OS

目的：ニュースから投資仮説を作り、検証し、判断履歴を残す一連の流れを完成させる。

- PWAとして利用可能
- AIなしでも動作
- AI深掘りは任意
- 日本株テーマに対応
- 仮説管理と簡易検証が可能
- 売買自動化はしない
