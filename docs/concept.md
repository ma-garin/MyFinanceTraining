# コンセプト

## プロダクト名

MyFinanceTraining

## 一文定義

世界のニュース・市場イベントを起点に、3〜5段階の連想ツリーを生成し、日本株/ETF/ETNの投資仮説を管理・検証する個人向け投資仮説OS。

## 背景

投資判断では、ニュースの一次反応だけでなく、生活者行動、企業行動、市場心理、資金ローテーションまで連想する必要がある。

例：

```text
中東情勢悪化
→ 原油供給不安
→ 移動費上昇
→ 外出抑制
→ 内篭り需要
→ ゲーム関連株が意識される
```

このような連想は、人間の直感で3段階目あたりに違和感や勝ち筋を見つけやすい。一方で、毎回すべてを手作業で展開・記録・検証するのは難しい。

## 目指すもの

- ニュースから投資仮説を作る
- 仮説を連想ツリーとして残す
- 期待方向だけでなく失敗条件も管理する
- 過去データで検証する
- 採用/保留/棄却の判断を蓄積する
- AIは補助として使い、ランニングコストを抑える

## 目指さないもの

- 売買の自動執行
- 投資助言サービス
- 常時AI APIを呼ぶ高コスト設計
- 単なる家計簿/資産管理アプリ
- 銘柄推奨だけを返すブラックボックスAI

## 基本思想

```text
LLM：連想・反論・見落とし検出
ルール：分類・テンプレ適用・スコア計算
人間：違和感検知・採否判断
データ：検証・棄却・再評価
```

## 対象市場

初期版では、出力対象を日本市場に絞る。

- 日本株
- 東証ETF/ETN
- J-REIT

入力情報は世界を対象にする。

- 米国株
- SOX/NASDAQ/S&P500/VIX
- 米雇用統計/FOMC/CPI
- 原油/金/米10年金利
- 為替
- 地政学ニュース
- 企業決算/ガイダンス
